# bs-pwa

Various bindings to browser APIs

## Install

```
npm i @zpnester/bs-filereader
npm i @zpnester/bs-pwa
```


* Notification
* ServiceWorkerRegistration
* HTMLAudioElement
* HTMLVideoElement
* MediaStream
* MediaStreamTrack
* Client
* WindowClient
* ServiceWorkerGlobalScope
* RTCPeerConnection
* RTCDataChannel
