# bs-pwa

Various bindings to browser APIs

## Install

```
npm i @zpnester/bs-filereader
npm i @zpnester/bs-pwa
```

## List of somewhat implemented bindings

* Notification
* ServiceWorkerRegistration
* HTMLAudioElement
* HTMLVideoElement
* HTMLCanvasElement
* MediaStream
* MediaStreamTrack
* Client
* WindowClient
* ServiceWorkerGlobalScope
* RTCPeerConnection
* RTCDataChannel
